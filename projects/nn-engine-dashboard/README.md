# 🖥️ NN Engine Dashboard

> 一个基于 Rust 后端 + 现代 Web 前端的网络引擎监控仪表盘

## 📋 项目简介

NN Engine Dashboard 是一个网络加速引擎的实时监控仪表盘，提供直观的 Web 界面用于监控网络连接、服务器状态、流量数据等关键指标。

### 主要特性

- **实时连接监控** — 展示总连接数、活跃连接数
- **流量统计** — 分别统计代理流量和直连流量
- **服务器管理** — 服务器列表展示，包含标签、协议类型、延迟、丢包率等
- **进程监控** — 进程数量实时统计
- **深色主题 UI** — 现代化的暗色主题界面设计

## 🎨 界面预览

仪表盘包含以下核心面板：

| 面板 | 功能 |
|------|------|
| 统计卡片 | 总连接数、活跃连接数、代理流量、直连流量、服务器数、进程数 |
| 服务器列表 | 标签、协议、延迟、丢包率、状态、TCP 流量等详细信息 |
| 实时状态 | 在线/离线/等待中的服务器状态实时更新 |

## 🚀 技术栈

- **后端**: Rust (NNAccEngine)
- **前端**: 原生 HTML5 + CSS3 + JavaScript
- **设计**: 深色主题，响应式布局
