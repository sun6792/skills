### 步骤2.5：公司风险验证（必须执行）

> **核心原则：匹配度再高的岗位，如果公司本身有风险，也必须降级或排除。每分析一个职位，必须同步进行公司背景核查。**

#### A. 公司工商信息核实

对每个分析到的公司，通过以下方式核查：

1. **天眼查**（tianyancha.com）/ **企查查**（qcc.com）/ **国家企业信用信息公示系统**（gsxt.gov.cn）
   - 使用 WebSearch 搜索 `"{公司全名} 天眼查"` 或 `"{公司全名} 企查查"`
2. **重点检查项：**
   | 检查项 | 正常 | 风险 |
   |--------|------|------|
   | 注册资本 vs 实缴资本 | 实缴≥注册资本50% | 实缴为0或极少 |
   | 参保人数 | 与公司规模匹配 | 参保人数为0-5（可能为空壳或全员外包） |
   | 经营状态 | 存续/在业 | 列入经营异常、严重违法 |
   | 成立时间 | ≥2年 | 不足半年（可能是新壳） |
   | 劳动仲裁 | 无或极少 | 多条劳动仲裁记录（拖欠工资信号） |
   | 行政处罚 | 无 | 多条行政处罚 |

#### B. 外包公司识别

1. **直接识别：** 看公司名称是否在上文列出的外包公司名单中
2. **JD关键词识别：** 售前岗位中出现以下词汇需高度警惕：
   - ❌ 高风险："驻场开发""外派""人力外包""劳务派遣""代招""外包岗位"
   - ⚠️ 需确认：有些小型安全集成商的售前岗需要常驻客户现场，需确认签约主体
3. **区分"外包派遣"和"渠道售前/厂家售前"：**
   - 厂家售前（✅ 最优）：奇安信/深信服/绿盟等直招，签约主体是安全厂商，薪资福利有保障
   - 渠道售前（⚠️ 需评估）：安全厂商代理商/集成商的售前岗，需确认代理商实力（总代/金牌/银牌级别）、代理产品线、是否有稳定客户群
   - 外包派遣（❌ 需排除）：签约主体是外包公司，以售前名义派到甲方

#### C. 骗子公司/虚假招聘识别

出现以下任一特征，标记为 🔴高风险：
- **培训贷陷阱：** JD或面试中提到"售前培训""网络安全培训""岗前培训后做售前""先培训后付费"
- **虚假高薪：** 薪资远超市场水平（应届生安全售前正常范围6-12K，承诺15K+且无方案经验要求需警惕）
- **收钱招聘：** 任何形式的收费（押金、培训费、服装费、体检费、工牌费）
- **办公地址异常：** 在居民小区、虚拟地址、无固定办公场所
- **公司刚注册：** 工商注册不到半年，无安全产品代理资质
- **面试流程异常：** 一轮面试即录用、面试官不像技术/售前人员、面试地点不在公司

#### D. 薪资保障评估

1. **员工口碑查询：**
   - 使用 WebSearch 搜索 `"{公司名} 拖欠工资"` `"{公司名} 工资 怎么样"` `"{公司名} 脉脉"`
   - 看准网（kanzhun.com）/ 脉脉搜索员工评价（薪资、出差补贴、报销是否及时、提成/奖金是否兑现）
2. **融资/经营状况：**
   - 安全厂商售前：通常薪资结构为底薪+项目奖金/提成，大型厂商底薪稳定
   - 小型安全集成商：需重点查经营状况，避免去了才发现公司回款困难影响工资
3. **面试时委婉确认：**
   - 可问："公司社保公积金是按实际工资缴纳吗？"
   - 可问："薪资结构是底薪+提成还是固定薪资？提成/奖金是按季度还是按年发放？"
   - 可问："出差补贴的标准是什么？报销周期是多久？"
   - 避坑：如果HR对薪资结构含糊或说"根据项目来""看业绩"，主动要求书面写明薪资构成

## 🛡️ 求职安全守则（每次搜索必须遵守）

### 投递前必查清单
每投递一个岗位前，确认以下事项：
- [ ] 已用天眼查/企查查核实公司工商信息（注册时间、实缴、参保人数、风险记录）
- [ ] 已搜索"{公司名} 拖欠工资"确认无负面记录（售前岗额外搜"{公司名} 提成 奖金"看是否兑现）
- [ ] 已看脉脉/看准网员工评价（售前岗重点看：报销是否及时、出差补贴是否到位、提成是否信守承诺）
- [ ] 确认签约主体不是外包/派遣公司（厂家直招✅，渠道商⚠️需评估，人力外包❌）
- [ ] 确认JD中无"培训后上岗""先培训再入职"等字眼
- [ ] 确认面试地点与工商注册地址一致
- [ ] 确认无任何形式的收费要求

### 面试安全提示
- **面试地点：** 只去公司办公场所面试，拒绝去酒店/咖啡厅/居民楼面试
- **收钱就跑：** 面试过程中任何形式的收费（培训费、押金、工装费等）都是骗局，立即终止
- **劳动合同：** 入职当天必须签劳动合同，看清楚签约主体公司名称、试用期时长、薪资结构（底薪+提成/绩效的构成和发放周期）与JD承诺是否一致
- **五险一金：** 问清楚缴纳基数和比例（按实际工资缴纳还是最低工资），入职第一个月就应开始缴纳
- **警惕模糊岗位名：** "安全顾问（管培生）""储备方案经理"等名称模糊的，可能不是真正的售前技术岗
- **薪资结构：** 售前岗常见薪资结构为底薪+项目提成/年终奖，问清楚：底薪多少？提成比例？奖金发放周期（月度/季度/年度）？上一年度同岗位平均拿到多少？拒绝"看项目来""看业绩"等模糊回答
- **出差待遇：** 售前岗出差频繁（30%-50%），问清楚：出差补贴标准（每天多少）、住宿交通标准、报销周期（月度or季度）、是否有调休

### 绝不投递的红线
出现以下任何一条，直接放弃该岗位：
1. 公司是外包/劳务派遣公司（非安全厂商/集成商正规售前编制）
2. JD或面试中提到"岗前培训""先培训"
3. 任何形式的收费
4. 工商状态异常（经营异常、严重违法、大量劳动仲裁）
5. 公司成立不到半年且查不到任何安全产品/代理资质
6. 有拖欠工资/报销/提成的明确记录
7. 岗位投递链接无法验证（404/内容不匹配/无法找到原始JD），但岗位本身信息可用——此时降级为参考，不写入"投递链接"
